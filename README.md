# Password Security Tool

## Description

This application is a desktop application developed in Java using JavaFX.
It helps users create secure passwords and manage authentication in a safe way.

The application allows users to:

* create an account
* log in using username and password
* check password strength
* generate secure passwords

Passwords are not stored in plain text, but are encrypted using the SHA-256 algorithm.

## Technologies Used

* Java
* JavaFX
* PostgreSQL
* JDBC
* Gradle


## Functionalities

* Sign Up (create account)
* Log In (authentication)
* Password strength verification (Weak / Medium / Strong)
* Password generator
* Show / Hide password
* Copy & Paste password
